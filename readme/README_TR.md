# ChatGPT Optimizer

![ChatGPT Optimizer Logo](../optimize.png)

ChatGPT Optimizer, ChatGPT web arayuzunde uzun sohbetlerde olusan hantalligi azaltmak icin konusma yanitini render oncesi optimize eder.


## Ne Yapar

- `https://chatgpt.com/*` ve `https://chat.openai.com/*` uzerinde calisir
- Ekranda son mesajlari tutarak uzun sohbetlerde akiciligi korur
- Varsayilanlarla kurulumdan sonra otomatik calisir
- Popup uzerinden simdi optimize et, daha eskileri goster ve hot reload kontrolu sunar

## Dil Davranisi

- Tarayici arayuzu `tr*` ise Turkce calisir
- Tarayici arayuzu `en*` ise Ingilizce calisir
- Tarayici dili Turkce veya Ingilizce degilse varsayilan olarak Ingilizce calisir

## Ne Yapmaz

- Sunucu tarafi gecikmeleri cozumlemez
- Veri toplamaz ve disariya veri gondermez

## Kurulum

1. Chrome'da `chrome://extensions` acin
2. `Developer mode` acin
3. `Load unpacked` ile bu klasoru secin
